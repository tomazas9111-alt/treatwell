# GrozioLaikas demo

Tai lokalus, be papildomu priklausomybiu veikiantis grozio salonu rezervacijos prototipas, ikveptas Treatwell tipo platformu.

## Kaip paleisti

1. Atidaryk `index.html` narsykleje.
2. Jei nori patogesnio testavimo, gali paleisti bet kuri statini failu serveri is sio katalogo.

## Kas jau veikia

- hero landing sekcija su paieska
- paslaugu kategoriju mygtukai
- salonu filtravimas pagal miesta, kategorija ir paieskos fraze
- rikiavimas pagal rekomendacija, reitinga ir kaina
- rezervacijos modalis su laiku pasirinkimu
- responsive dizainas mobiliesiems ir desktop
